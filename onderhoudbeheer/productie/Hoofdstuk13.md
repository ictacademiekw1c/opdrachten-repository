# Hoofdstuk 13 Acceptatietest (B1-K3)

## 13.0 Algemeen

De acceptatietest is de stap voordat een ontwikkelde informatiesysteem ook echt wordt opgeleverd.
De applicatie moet nog worden getest door een testgroep van eindgebruikers.

Voordat de test kan worden uitgevoerd moet er eerst een testplan worden opgesteld.

Een manier om een testplan op te stellen die door een eindgebruiker kan worden uitgevoerd is door alle functionaliteiten die in het functioneel/technisch ontwerp reeds zijn genoemd deze structureel en per onderdeel in het testplan op te nemen.



## 13.1 Situatie / Case Koninklijk Concertgebouw

## Opdracht 1

Je stelt een acceptatietest op voor de functionaliteiten uit de applicatie. 

__Testplan sjabloon__

Als sjabloon voor het testen van de applicatie kan het volgende bestand worden gebruikt:

<a href="https://elo.kw1c.nl/CMS/Studie/811%20ICT-Academie/811%20VakkenInhoud/%5BB.06%20BEH%5D%20Onderhoud%20en%20beheer/Productie/04.%20Aanvullend/Sjabloon%20Acceptatietest.docx">sjabloon testplan</a>


Je maakt gebruik van alle materiaal uit het volgende <a href="https://elo.kw1c.nl/CMS/Studie/811%20ICT-Academie/811%20VakkenInhoud/%5BB.06%20BEH%5D%20Onderhoud%20en%20beheer/Productie/04.%20Aanvullend/Acceptatietest%20Concertgebouw.rar" target="_new">zip bestand.</a>
In het zip bestand vind je:

Wat nog ontbreekt is een sql script om de database te vullen met een minimale content zodat de applicatie goed getest kan worden.
Zorg dus eerst voor een goede initiele vulling van de database. Schrijf dan je acceptatietestplan aan de hand van het functioneel ontwerp.

## Opdracht 2

Jij gaat een eindgebruiker begeleiden bij het uitvoeren van de acceptatietest. 
 
Uitvoeren acceptatietest 
    - Begeleid de gebruiker bij het uitvoeren van de acceptatietest. 
        a. Leid de gebruiker door de verschillende onderdelen van de acceptatietest. 
        b. Maak aantekeningen tijdens het uitvoeren van de test. 
    - Werk de testresultaten van de acceptatietest uit in je Acceptatietest document die je in Opdracht 1 gedeeltelijk hebt ingevuld. Je vult nu de regel Werkelijk resultaat in.
